\subsection{Detailed Description}
Stores the database of poses for Pose Estimation. 

This class is used internally by \hyperlink{classPoseEstimation}{Pose\-Estimation}, however it can be used to create or load multiple databases and test pose estimation with them. I.\-e. with set\-Database method of \hyperlink{classPoseEstimation}{Pose\-Estimation}. \begin{DoxyAuthor}{Author}
Federico Spinelli 
\end{DoxyAuthor}
